# **App Name**: Travex

## Core Features:

- Intelligent Expense Capture: Quick-entry wizard for expenses featuring a smart auto-categorization tool that analyzes titles to predict categories.
- Hybrid Group Participants: Manage trips with both registered app 'Friends' and non-user 'Members' (like family groups) stored in the database.
- Dynamic Expense Splitting: Multi-mode splitting logic supporting equal, group-based, and custom individual weight distributions.
- Live Trip Balances: Real-time ledger updates reflecting exactly who owes whom across mixed currency transactions.
- Real-time Trip Feed: A chronological view of all shared costs with interactive status chips and category visual cues.
- Automated Category Mapping: A backend service tool that leverages patterns to sort transaction keywords into expense groups automatically.
- Multi-Provider Auth: Seamless onboarding using Google Sign-In and secure Email/Password authentication through Firebase.
- Responsive Web and Mobile Access: Cross-platform accessibility ensuring users can manage expenses on both native mobile apps and a web dashboard.

## Style Guidelines:

- Primary Brand: Ocean Teal (#0B6E82) for main brand color, active navigation, and primary actions.
- Headers and Text: Deep Navy (#112240) for app bars, headers, primary text, and dark surfaces.
- Call to Actions: Amber Orange (#F5A623) for CTAs, FABs, highlights, and active states.
- Backgrounds and Surfaces: Cream White (#F2EFE4) for warm backgrounds with pure White (#FFFFFF) for cards, sheets, and elevated surfaces.
- Supporting Palette: Muted Teal (#5B9EAD), Dark Teal (#084F5E), Navy Muted (#2C4A6B), Primary Light (#D6EBF0), and Accent Light (#FEF0D3).
- The 'DM Sans' font family provides a clean, geometric, and modern aesthetic for all textual content.
- Use 'Iconsax' outline/linear style icons exclusively to maintain consistency and a refined visual identity.
- A responsive mobile and web layout system with a 20px horizontal padding and 16px card border radius.
- Utilize Lottie animations for empty states and SharedAxis horizontal transitions for smooth wizard navigation.
